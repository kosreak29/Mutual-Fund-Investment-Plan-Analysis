# Mutual Fund Investment Plan Analysis

## Project Overview
This project focuses on creating a **Mutual Fund Investment Plan using historical stock market data**. The objective is to identify stocks that provide **high returns with relatively low risk**.

Using Python and data analysis techniques, the project calculates key financial metrics such as **Return on Investment (ROI)** and **Volatility** to evaluate stock performance. Based on these metrics, a portfolio of selected stocks is created to simulate a **mutual fund strategy for long-term investment**.

---

## Problem Statement
Mutual funds pool money from multiple investors and invest it in a diversified portfolio of stocks and other securities.

The goal of this project is to:

- Analyze historical stock data
- Identify stocks with **high ROI and low volatility**
- Create a **balanced investment portfolio**
- Simulate the **future value of monthly investments**

---

## Objectives

The main objectives of this project are:

1. Load and explore historical stock price data.
2. Calculate **Return on Investment (ROI)** for each company.
3. Measure **Volatility (Risk)** using daily returns.
4. Select stocks that provide **high returns with lower risk**.
5. Allocate portfolio weights using **inverse volatility allocation**.
6. Compare the portfolio with high-growth stocks.
7. Simulate investment growth for **1 year, 3 years, 5 years, and 10 years**.
8. Visualize portfolio performance and investment growth.

## Key Metrics Used

### Return on Investment (ROI)

ROI measures the total return generated by a stock over a period of time.

**Formula**

ROI = (Final Price − Initial Price) / Initial Price × 100

Higher ROI indicates better performance.

---

### Volatility (Risk)

Volatility measures the **risk or fluctuation in stock prices**.

It is calculated as the **standard deviation of daily returns**.

Higher volatility means higher risk.

---

## Portfolio Strategy

To build the mutual fund plan:

1. All stocks were ranked based on **ROI and volatility**.
2. Stocks with **high ROI and lower volatility** were selected.
3. Portfolio weights were assigned using **inverse volatility allocation**.

This means:

- Stocks with **lower volatility receive higher weight**
- Stocks with **higher volatility receive smaller weight**

This strategy helps balance **risk and return**.
